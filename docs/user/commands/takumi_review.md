## takumi review

Run a thorough code review of all workspace changes

### Synopsis

Analyze all uncommitted or branched changes in the workspace using an AI
sub-agent. The agent can read files, run commands, and explore the codebase
to produce a thorough code review document.

Supports multiple LLM providers. Set one of:
  ANTHROPIC_API_KEY  — uses Anthropic (Claude models)
  OPENAI_API_KEY     — uses OpenAI (GPT models)

Or specify explicitly with --provider and --model.

```
takumi review [flags]
```

### Options

```
      --base string       base ref for diff (e.g. main, HEAD~3) (default "HEAD")
  -h, --help              help for review
      --max-turns int     maximum agent turns (default 20)
      --model string      LLM model (default: auto-detected from provider)
  -o, --output string     write review to file (default: .takumi/reviews/<timestamp>.md)
      --provider string   LLM provider: anthropic, openai (default: auto-detected from env)
```
