## takumi test

Run test phase for packages

### Synopsis

Run tests for packages in dependency order. If no packages are specified,
tests all packages in the workspace. Use --affected to test only changed
packages and their downstream dependents.

```
takumi test [packages...] [flags]
```

### Options

```
      --affected   Only test packages affected by changes
      --dry-run    Show execution plan without running anything
  -h, --help       help for test
      --no-cache   Skip cache and force execution
```
