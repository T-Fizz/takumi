## takumi build

Run build phase for packages

### Synopsis

Build packages in dependency order. If no packages are specified, builds all
packages in the workspace. Use --affected to build only changed packages
and their downstream dependents.

Subcommands:
  clean    Remove build artifacts

```
takumi build [packages...] [flags]
```

### Options

```
      --affected   Only build packages affected by changes
      --dry-run    Show execution plan without running anything
  -h, --help       help for build
      --no-cache   Skip cache and force execution
```
