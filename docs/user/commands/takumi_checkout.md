## takumi checkout

Clone a repo and add it to the workspace

### Synopsis

Clone a git repository into the workspace and register it as a tracked source.
Detects takumi-pkg.yaml in the cloned repo and sets up runtime environments.
If the repo contains sub-packages, all are registered.

```
takumi checkout <clone-url> [flags]
```

  -h, --help            help for checkout
      --path string     Local path for the cloned repo
