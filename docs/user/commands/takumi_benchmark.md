## takumi benchmark

Run performance benchmarks comparing agent work with and without Takumi

### Synopsis

Run identical tasks with and without Takumi operator instructions, measuring
token usage, tool calls, turns, errors, and wall-clock time. Optionally publish
results to a shareable GitHub Gist dashboard.

Available scenarios: fix-build-error, scoped-rebuild, understand-structure
If no scenarios specified, all are run.

Subcommands:
  iterate    Run iterative setup benchmark and track improvements over time

```
takumi benchmark [scenarios...] [flags]
```

      --publish        publish results to a GitHub Gist dashboard
